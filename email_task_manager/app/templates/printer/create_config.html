Printer Setup Guide

Configure your receipt printer for task printing. Fields marked with * are required.

Printer Types:
  • CUPS: For Mac and Linux systems
  • Windows: For Windows systems
  • Network: For network-connected receipt printers
  • USB: For directly connected USB receipt printers
  • File: Output to a file (for testing)
Tips:
  • For Windows/CUPS printers, use the exact printer name from your system
  • For network printers, provide the IP address and port
  • For USB printers, provide the vendor and product IDs (e.g., "0x1234:0x5678")
  • For file output, provide the full path to the output file
  • Test your configuration after saving
After creating the printer configuration, you can test it by printing a test page.
